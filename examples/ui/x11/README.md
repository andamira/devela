# `ui_x11` example

This example demonstrates devela's UI abstractions using an X11 backend.

It is intended to evolve alongside the corresponding UI examples for other
backends while preserving a comparable overall structure.

## Requirements

On Debian-based systems:

```sh
sudo apt install libxcb1-dev libxkbcommon-x11-dev libxkbcommon-dev
```

## Run

Run the example from this directory:

```
cargo run
```

Press Q or close the window to exit.
